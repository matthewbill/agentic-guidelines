# Data Table Guidelines

## First Columns

The first columns of any table should be:

1. Expander/Collapser (for child rows)
2. A checkbox which appears when you hover over it
3. An avatar (where appropriate) and name of entity which takes you to the page when you click on it.

## Last Columns

The last columns should be (left to right)

- Quick Actions
- More Actions
- Open Details Panel
