# Visual Signifier Guidelines

Design elements that communicates purpose or meaning through appearance.

| Shape           | Type        | Examples                                       |
| --------------- | ----------- | ---------------------------------------------- |
| Rounded squares | Collections | groups, teams                                  |
| Circles         | Items       | Applications, Users, Products                  |
| Hexagons        | Components  | Individual parts, components, services, blocks |
| Pills           | Tags        | Metadata, labels                               |
