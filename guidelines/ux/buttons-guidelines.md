# Bustton Guidelines

Default (Primary)

- Use for primary actions that stand out the most.
- Example: "Submit," "Save," "Continue," "Next."

Secondary

- Use for secondary actions that are still important but not the main focus.
- Example: "Back," "View Details."

Destructive

- Use for dangerous actions that have a serious consequence.
- Example: "Delete," "Remove," "Revoke Access."

Link

- Use for navigation, replacing traditional anchor (`<a>`).
- Ideal for links styled as buttons without strong visual emphasis.
- Example: "Read More," "Go to Documentation."

Outline

- Use when an action sis optional or non-disruptive.
- Good for actions that don't require trong emphasis.
- Example: "Learn More," "Preview," "Expand.", "Cancel"

Ghost

- Use for subtle actions, especially when placed in a toolbar or navigation bar.
- Great for non-intrusive interactions.
- Example: "Settings," "Help," "View Profile."
